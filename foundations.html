Token Architecture

Carbon의 역할 기반 토큰과 Primer의 3단계 모델에서 영감. 하드코딩된 hex 값을 토큰으로 대체한다.

Base

원시 컬러 값. UI에서 직접 사용 금지.

palette-gray-12

기능 토큰에서만 참조

Functional

Role-based tokens for backgrounds, text, borders.

--color-bg-default

Primary usage in components

Component

Scoped tokens for specific component states.

--color-surface-card

Override when functional tokens are insufficient

Do — 테마 호환성을 위해 시맨틱 토큰 클래스 사용
className="bg-background text-foreground"
Don't — 하드코딩된 hex 값 금지 — 테마 변경 시 깨진다
className="bg-[#0A0A0A] text-[#FFFFFF]"
