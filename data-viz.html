Chart Design Rules

모든 차트에 일관되게 적용하는 6가지 규칙. 코드 리뷰 체크리스트로도 사용한다.

6가지 규칙 (C1–C6)

규칙 ID는 코드 주석과 리뷰에서 그대로 참조한다.

C1

Y축 그리드라인: border-white/[0.04]

차트 배경을 방해하지 않으면서 눈금 위치를 안내한다.

C2

Data label은 hover(tooltip)에서만 표시

항상 표시하면 데이터 포인트가 많을 때 겹쳐 읽기 어렵다.

C3

범례(Legend)는 상단 또는 우측. 하단 금지.

하단 범례는 시선이 차트 → 범례 → 차트로 분산된다.

C4

축 라벨: text-xs text-white/30

축 라벨은 보조 정보이므로 본문보다 작고 흐리게 처리한다.

C5

Data point(점) 최소 크기: 8px

8px 미만은 터치 디바이스에서 hover/tap이 어렵다.

C6

차트 최소 높이 200px, 권장 280–320px

200px 미만은 데이터 밀도가 높아질 때 판독이 불가능해진다.

스타일 데모

위 규칙을 모두 적용한 CSS-only 차트 목업. 실제 구현 시 Recharts 등 라이브러리를 사용하되 동일한 스타일 규칙을 따른다.
