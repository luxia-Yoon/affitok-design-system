Motion

마이크로 인터랙션과 애니메이션. GPU-accelerated, 60fps.

Motion Style

Carbon에서 영감받은 2가지 모션 스타일: Productive는 태스크 중심 UI용, Expressive는 중요한 순간용.

Productive Motion

Efficient, subtle, out of the way. For microinteractions: button states, dropdowns, tooltips, data tables.

CurveCSS
Standardcubic-bezier(0.2, 0, 0.38, 0.9)
Entrancecubic-bezier(0, 0, 0.38, 0.9)
Exitcubic-bezier(0.2, 0, 1, 0.9)

Expressive Motion

Enthusiastic, vibrant, visible. For significant moments: page transitions, primary CTA, notifications, alerts.

CurveCSS
Standardcubic-bezier(0.4, 0.14, 0.3, 1)
Entrancecubic-bezier(0, 0, 0.3, 1)
Exitcubic-bezier(0.4, 0.14, 1, 1)
Do — Fast, purposeful motion. Productive for common interactions.

Button hover: 100ms ease-out

Modal enter: 200ms cubic-bezier(0,0,0.3,1)

Don't — Bouncy, slow, or linear motion feels unnatural and distracting.

Button hover: 500ms bounce

Modal enter: 1000ms linear

Easing Curves

화살표나 트랙을 클릭하면 재생. 3가지 주요 커브로 모든 UI 모션을 커버한다.

ease-out

대부분의 UI 인터랙션 기본값. 요소가 감속하며 멈춘다.

ease-out
spring

토글, 바운스, 재미있는 순간을 위한 오버슈트 효과.

cubic-bezier(0.34, 1.56, 0.64, 1)
expo-out

부드럽고 드라마틱한 등장. 페이지, 모달, 패널.

cubic-bezier(0.16, 1, 0.3, 1)

Easing Decision Tree

어떤 커브를 언제 사용하는지.

ScenarioEasingExample
Element visible start to endStandard (ease-out)Sort table, expand tile
Element enters viewEntrance (expo-out)Modal appear, toast slide-in
Element exits view permanentlyExit (ease-in)Modal close, toast dismiss
Element exits but stays nearbyStandard (ease-out)Side panel close (can reopen)
Toggle or bouncy feedbackSpringSwitch toggle, like heart, star

Button States

호버: translateY(-1px) 떠오름. 액티브: scale(0.98) 눌림. GPU 가속 transform만 사용.

HOVER LIFT + ACTIVE PRESS

IMPLEMENTATION

/* Hover: Anticipation (Disney #2) */
.btn { transition: all 100ms ease-out; }
.btn:hover {
  transform: translateY(-1px);
  box-shadow: 0 8px 25px rgb(var(--primary) / 0.25);
}

/* Active: Squash & Stretch (Disney #1) */
.btn:active {
  transform: translateY(0) scale(0.98);
  box-shadow: 0 2px 8px rgb(var(--primary) / 0.1);
}

Toggle & Checkbox

토글은 spring 이징, 체크마크는 stroke-dashoffset 사용.

SPRING TOGGLE

transition: transform 200ms
  cubic-bezier(0.34, 1.56, 0.64, 1);
/* spring overshoot — bouncy feel */

STROKE CHECKBOX

stroke-dasharray: 20;
stroke-dashoffset: 0;   /* checked */
transition: stroke-dashoffset
  200ms ease-out 50ms;  /* 50ms delay */

Performance Rules

GPU 합성 속성만 애니메이션한다. 레이아웃 스래싱을 피한다.

PropertyLayerPerformance
transformCompositeBest — GPU accelerated
opacityCompositeBest — GPU accelerated
filterCompositeGood — GPU on most browsers
background-colorPaintOK — repaint only, no reflow
box-shadowPaintExpensive — avoid animating
width / heightLayoutWorst — triggers reflow
Do — Animate transform and opacity — they run on the GPU compositor
transform: translateY(-1px) scale(0.98)
Don't — Animating margin/width triggers layout recalculation every frame
margin-top: -1px; width: 98%
